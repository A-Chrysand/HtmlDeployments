输入框:

输入普通文本的框框，如账号：

  • <input type="text"/>
样例：文本输入框

输入不给看的文本的框框，如密码：

  • <input type="password"/>
样例：密码输入框

上面的输入框，都可以添加以下属性来提高可读性：

属性名 属性含义
name 输入框的名称(一般用于js控制)
value 输入框的数值类型(如数字型和字符型)
placeholder 在框中加入提示

选择框:

单选框：

  • <input type="radio"/>
样例：选择框： 选项1 选项2

多选框：

  • <input type="checkbox"/>
样例：多选框： 选项1 选项2 选项3 选项4

上面的选择框，都可以添加以下属性来提高可读性：

属性名 属性含义
name 输入框的名称，选择框的名称一定要相同
checked 添加提前勾选选项

下拉选项表：

  1. <select>
  2. <option value="c1"选项1</option>
  3. <option value="c2"选项2</option>
  4. <option value="c3"选项3</option>
  5. </select>
  6. 选项将从上到下依次排序

若需设置默认显示，只需要在<option>里添加selected属性

文本输入框:

输入较多文本的框框，如评论、感想、回贴等：

  • <textarea></textarea>

可以在<textarea>中添加[maxlength="自定义数值"]属性来控制最大字符输入量
