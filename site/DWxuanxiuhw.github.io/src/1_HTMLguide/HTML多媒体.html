音频：

在HTML5中，内置了音频播放器，可以通过<audio></audio>标签来添加音频

  1. <audio>
  2. <source src="./audio/music.mp3" type="audio/mp3" />
  3. <source src="./audio/music.wav" type="audio/wav" />
  4. <source src="./audio/music.ogg" type="audio/ogg" />
  5. <embed height="100 width="100 src="song.mp3" />
  6. </audio>
  7. HTML5<audio>元素会尝试以mp3、wav或ogg格式来播放音频，如果失败，代码将会回退尝试<embed>元素
  8. <embed>元素定义外部内容的容器，能够讲音频嵌入网页中

Audioplay还能增加一些属性：

属性名 属性含义 举例
autoplay 自动播放 autoplay="autoplay"
controls 播放控件
loop 循环播放 loop="loop"
muted 静音
preload 预加载播放

也可以采用外链雅虎播放器：

  1. <a href="./audio/song.mp3">点击播放</a>
  2. <script type="text/javascript" src="http://mediaplayer.yahoo.com/js"></script>

视频：

在HTML5中，同样内置了视频播放器，可以通过<video></video>标签来添加视频

  1. <video width="550" controls="controls">
  2. <source src="./video/Sail-Away.mp4" type="video/mp4" />
  3. <source src="./video/Sail-Away.ogv" type="video/ogv" />
  4. <source src="./video/Sail-Away.webm" type="video/webm" />
  5. <object data="./video/Sail-Away.mp4" width="320" height="240">
  6. <embed src="./video/Sail-Away.swf" width="320" height="240" />
  7. </video>
  8. HTML5<audio>元素会尝试以 mp4、ogg 或 webm 格式中的一种来播放视频。如果失败，代码将会回退尝试<embed>元素

插入视频的方式：

1、采用<object>的方式插入：

  • <object data="视频文件地址"></object>
  • <object>是把多媒体文件当作对象来处理，它可以插入几乎所有的多媒体对象。

2、采用<embed>的方式插入：

  • <embed src="视频文件地址"></embed>
  • <embed>是以前所广泛应用的插入方式，它可以插入网页、视频、音频，简直为所欲为。但是它是使用Flash作为接口，抵制中国特供Flash从你我做起！

3、采用<iframe>的方式插入：

  • <iframe src="视频文件地址"></iframe>
  • <iframe>是HTML5所广泛应用的插入方式，它还可以用来插入网页。
